## Quick Reference

- **App name:** Travel KSA
- **Package:** `com.example.travelaks`
- **Firebase project:** `travel-ksa-meta`
- **Admin login:** `admin` / `admin123` (local, not Firebase Auth)
- **Platform:** Android (Java), minSdk 24, targetSdk 36
- **Database:** Cloud Firestore (Firebase)
- **Auth:** Firebase Authentication (Email/Password)
